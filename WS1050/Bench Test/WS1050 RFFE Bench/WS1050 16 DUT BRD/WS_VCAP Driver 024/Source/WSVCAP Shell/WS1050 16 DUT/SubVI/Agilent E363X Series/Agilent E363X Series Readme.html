Agilent E363X Instrument Driver Readme

To rate this driver, go to the driver download page on the Instrument Driver Network.

1. Overview

Instrument Driver Technology: LabVIEW Plug and Play (project-style)
Manufacturer: Agilent Technologies
Supported Language(s): LabVIEW 8.0
Supported Model(s): Agilent Technologies E3631A, E3632A, E3633A, E3634A
Model(s) Tested: Agilent Technologies E3631A, E3632A
Interface(s): GPIB, Serial
Firmware Revision Tested: 2.1-5.0-1.0

Certified: Yes
NI Supported: Yes
Source Code Available: Yes

Driver Revision: 3.1.1
Original Release Date: 9/20/2005
Current Revision Date: 07/01/2009

2. Required Software

Some software components need to be installed before using this instrument driver. The minimum versions of these components are listed below, and can be downloaded from the Download Site.

VISA 3.0 or later
LabVIEW 8.0 or later


For more details, refer to the LabVIEW Help under Help » VI, Function, & How-To Help.

3. Known Issues

To report issues or provide feedback about this instrument driver, please send an email to instrument.drivers@ni.com.

4. Revision History

The latest version of this and other LabVIEW and LabWindows/CVI instrument drivers can be downloaded at the Instrument Driver Network.

REV 3.00, 9/20/05
Modified by: Rob Hammond, Austin,TX.
Created new driver from hpe363xa version 2.0 to be shipped with LabVIEW 8.0.

REV 3.0.1 5/1/07
Modified by Nathan Tacha, Austin TX.
Changed Read Output.vi to remove a VISA write that was passing an empty string constant

REV 3.0.2 11/14/08
Modified by Nathan Tacha, Austin TX.
Added a 100 ms Wait to the serial case of Initialize VI to give the instrument ample time to respond also changed Flow Control to None

REV 3.0.3 11/19/08
Modified by Nathan Tacha, Austin TX.
Added a 100 ms Wait after the VISA Write in Reset VI when using serial and added a 500 ms wait after the VISA write in Default Instrument Setup VI when using serial

REV 3.1 12/19/08
Modified by Nathan Tacha, Austin TX.
Changed the Serial Flow Control from a constant to a control and connected it to the connector pane in Initialize VI. This is so the user can change the Flow Control to correspond to the instrument's current setting.

REV 3.1.1 07/01/09
Modified by Alvin Tang, Austin TX.
Added time delay for serial interface in Read Output VI, Error Query VI and Examples to give the instrument enough time to respond.
