Keithley 2450/2460/2461 LabVIEW Instrument Driver Readme

1. Overview

Instrument Driver Technology: LabVIEW Plug and Play (project-style)
Manufacturer: Keithley
Supported Language(s): LabVIEW 2012 and newer
Supported Model(s): 2450, 2450-NFP, 2460, 2461
Model(s) Tested: 2450, 2460, 2461
Interface(s): GPIB, USB, LAN(VXI-11)
Firmware Revision(s) Tested:1.4.1d

Certified: No
NI Supported: No
Keithley Supported: Yes
Source Code Available: Yes

Driver Revision: 1.3.0.0
Original Release Date: 8/14/2013
Current Revision Date: 1/27/2016

2. Required Software

Some software components need to be installed before using this instrument driver. The minimum versions of these components are listed below, and can be downloaded from the Download Site.
VISA 3.2 or later

Refer to the LabVIEW Help for more information about software requirements. You access the LabVIEW Help by selecting Help»Search the LabVIEW Help.

3. Known Issues

If using the Configure Pulse List Sweep.vi, you cannot have the source auto range. Use a fixed source range using the Configure Source Range.vi instead.

To report issues or provide feedback about this instrument driver, please visit the Keithley forums at: http://forum.keithley.com

4. Revision History

REV 1.1.0.0, 8/14/2013
Original Release

REV 1.1.1.0 8/26/2013

PR50553: Set IO Pins vi was removed. Functionality available in Write to all Digital IO Pins vi.

 

PR50554: Configure Limit Test Settings vi removed from driver as command no longer supported.

 

PR50555: Updated Configure Linear Step Mode Sweep vi default values for start, stop and step. Also updated documentation for step control to clarify this is for voltage/current step and not the amount of steps in the sweep.

 

PR50556: Configure Log Mode Sweep vi has been updated with default Start value of 0.1 instead of 0 to prevent asymptote issue.

 

PR50557: Configure Measurement Auto Range Lower Limit vi has updated its Manual Range control default value to 0.1.

 

PR50558: Added Configure Measurement Auto Range Upper Limit vi.

 

PR50559: The Configure Reading Format of Display vi had the AUTO option removed as it is not supported.

 

PR50560: Configure Trigger Model Constant Delay Block vi has the Delay in seconds control updated from 1us to 50us.

 

PR50561: The Load Trigger Model Configuration List vi has a SCPI syntax error that has been resolved.

 

PR50562: The Abort vi had SCPI syntax error that is now resolved.

 

PR50563: The Beeper Enable vi was removed from the driver as that command is no longer supported.

 

PR50564: The Get Display Screen vi was removed as it is no longer supported.

 

 

REV 1.1.2.0 1/26/2014

PR50747: Renamed the Resistance measurement function to Auto Resistance. Added a Manual Resistance measurement function.

 

PR50947: Palette files for LabVIEW version 8 have been repaired.


REV 1.2.0.0 09/08/2014

Added model support for 2460 series Instruments.

Help content updated for several VIs and Controls.


REV 1.3.0.0 01/27/2016

Added model support for 2461 series Instruments.

Help content updated for several VIs and Controls.
